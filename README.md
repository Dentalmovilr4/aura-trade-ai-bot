Aura Trade AI Bot 🚀

Sistema de trading automatizado con inteligencia artificial que genera señales en tiempo real, entrena modelos y permite validación mediante backtesting.

🔥 Features

- Datos reales del mercado
- IA con RandomForest
- Backtesting integrado
- Envío de señales automático
- Dashboard visual (Streamlit)

⚙️ Instalación

git clone https://github.com/tu-usuario/aura-trade-ai-bot.git
cd aura-trade-ai-bot
pip install -r requirements.txt

▶️ Ejecutar bot

python src/main.py

📊 Dashboard

streamlit run dashboard/dashboard.py
